# PF_LOX Documentation

PF_LOX is a tree-walk interpreter for a functional variant of the lox
programming language written in python. Inspired by the book "Crafting interpreters."

This documentation is organized as follows:

- Architecture - describing the high level structure and design of the interpreter
- Syntax - specifying the lexical and syntactic grammar of the language
- Language - specifying the semantic details of the language
- Interpreter - explaining the implementation details of this interpreter

Example code, and the test suits are in the top level directories examples/ and tests/
Source code is located in the src/ directory.
